# aai-510-finalproject-group-4

# Features

This project builds an AI agent to analyze application user feedback and sentiment.

### Main features:

- Search user reviews by app, category, or keyword
- Summarize large amounts of review content
- Identify common user pain points
- Analyze user sentiment and feedback trends
- Compare feedback across different AI applications
- Generate product insights from user reviews

### How to Run

`This project is designed to run in Databricks Notebook.
`
